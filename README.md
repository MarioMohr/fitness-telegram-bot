# Fitness Tracker Telegram Bot

A containerized Telegram bot system built to track daily fitness activities, swimming volume, nutrition logs, and weight statistics. 

## Project Structure
* `app/` - Contains the primary Python application logic and bot interface scripts.
* `data/` - Local storage mount point for the tracking database.
* `docs/` - System documentation, installation workflows, and architecture references.

## Core Features
* Clean interactive custom navigation menus.
* On demand logging for specific workout selection menus.
* Redundant system backup distribution pipelines targeting mounted external drives.
